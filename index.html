As a group project, we developed an IoT-based automatic parking spot finder using an ESP32-CAM for image capture. The system employs computer vision with YOLOv8 to detect available parking spaces, transmitting the data via MQTT to a local server (Raspberry Pi 5) for web-based monitoring."

Bytez

Matlab, QOS, Wireshark

Developed a standalone MATLAB application to analyze and calculate Quality of Service (QoS) metrics for ICMP packets by processing data from Wireshark CSV files. This application is capable of parsing ICMP packet data from these files to evaluate network performance parameters such as latency, jitter, and packet loss, thereby providing a comprehensive QoS analysis.
