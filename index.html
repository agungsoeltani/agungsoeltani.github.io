Selected Works

Flow

An IoT-based automatic fan controller utilizing a DHT22 sensor, enabling real-time environmental monitoring and fan operation control via a web interface using the MQTT protocol.

Sbite

Exploration and design of a WiFi and Bluetooth signal jammer device to understand interference and frequency spectrum security.

Smapar

Developed an IoT-based automatic parking spot finder using an ESP32-CAM for image capture. The system employs computer vision with YOLOv8 to detect available parking spaces, transmitting the data via MQTT to a local server (Raspberry Pi 5) for web-based monitoring

Yagi Antenna

Designed and simulated a Yagi antenna in CST Studio for TV broadcasting applications, operating in the 400-600 MHz frequency range, with a focus on optimizing gain and radiation patterns.
